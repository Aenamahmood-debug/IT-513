Web Development Glossary

HTML
HyperText Markup Language used to structure content on the web.
CSS
Cascading Style Sheets used for styling HTML content.
JavaScript
A programming language used to create dynamic web pages.
DOM
Document Object Model, an interface for HTML and XML documents.
API
Application Programming Interface allows software to interact with other software.
Responsive Design
Design approach that ensures websites work on all devices.
SEO
Search Engine Optimization improves a site’s visibility on search engines.
Accessibility
Designing websites so people with disabilities can use them effectively.
